## 🔄 How Area-Based Routing Works

When a citizen files a complaint:
1. NLP classifies the complaint → determines department (e.g. GHMC, BWSSB)
2. The complaint's `district_id` is matched against officers who have the same `district_id`
3. If a matching officer exists for that dept + district → auto-assigned to them
4. If no area match → falls back to any officer in that department

To run the city officers seed:
```bash
# First seed location hierarchy (if not done)
npm run seed:locations

# Then seed city officers
npm run seed:officers
```
